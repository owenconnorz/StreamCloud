# StreamCloud

Modern plugin streaming client for Android.

Built for fast streaming, modern UI, and seamless media playback with support for providers, Stremio-style addons, RealDebrid integration, casting, and Material 3 design.

---

## Features

### Streaming
- Stream movies and TV shows instantly
- Stremio addon/provider support
- Nuvio plugins
- Cloudstream plugins
- Multiple source support
- Fast source loading
- RealDebrid compatibility
- Auto-next episode
- Continue watching
- Subtitle support
- Multiple video qualities
- External player support

### Media Player
- Media3 / ExoPlayer
- Chromecast / Google Cast support
- Picture-in-picture support
- Custom playback controls
- Gesture controls
- Playback speed controls
- Resume playback
- Buffer optimization

### Library & Discovery
- Trending movies & TV shows
- Search support
- Personalized recommendations
- Watch history
- Favorites & bookmarks
- Continue watching row
- TMDB metadata integration
- Anime support

### Interface
- Material 3 UI
- Dynamic colors
- AMOLED dark theme
- Smooth animations
- Mobile optimized layouts
- Netflix-style design

### Performance
- Optimized streaming pipeline
- Fast caching
- Lightweight architecture
- Modern Kotlin codebase
- Coroutines & Flow
- Responsive UI

## Installation

Clone the repository:

```bash
git clone https://github.com/owenconnorz/StreamCloud.git
```

Open the project in Android Studio and build:

```bash
./gradlew assembleDebug
```

---

## Development

### Requirements
- Android Studio
- JDK 17+
- Android SDK
- Gradle

### Debug Build

```bash
./gradlew assembleDebug
```

### Release Build

```bash
./gradlew assembleRelease
```
